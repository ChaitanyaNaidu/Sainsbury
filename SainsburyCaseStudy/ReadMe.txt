How to execute the code:

Note: it is a Maven project and have couple of dependencies.
1. Download the project 'SainsburyCaseStudy'
2. Import the project in IntelliJ/eclipse.
3. Go to /src/test/java/com/sainsbury/ripefruit/RunMe.java
4. Right click on 'getJsonResponse' method and run as standalone.
5. That will assert and print the response in console.
